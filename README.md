launchcraft
===========

A python script to automate the installation of my preferred minecraft setup.

Prerequisites
=============

* Linux
* zip
* python 2.7
  * virtualenv
  * setuptools

Installation
============

Ensure you have run minecraft at least once and installed `1.7.2`.

    make venv

Then update `config.py` with your information.

Running
=======

    make
